ANALYSIS MODE: MINIMUM STRATEGY (FACT EXTRACTION / NORMALIZATION)

Definition:
A header is ONLY a cell that represents an INDEPENDENT ATTRIBUTE or ENTITY required to express the table in normalized (relational) form.

Core principle:
The table must be conceptually transformed into a normalized structure:

— identify entities (objects)
— identify their attributes
— separate structure from raw values

Headers are ONLY the elements that define the structure (schema) of the data.
